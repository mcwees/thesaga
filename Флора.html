Профессия Флора подразделяется на три навыка:

Ботаника Сбор растений, которые нужны для производства зелий с помощью навыка Травничество.
Садоводство Сбор урожая, который используется другими навыками, такими как: Дрессировка ( Яблоки для приручения животных) и Портняжное дело ( Вязанки культур для обработки их в нити).
Травничество Изготовление зелий, красок и кормов, которые нужны для прокачки навыков у спутников-животных.
