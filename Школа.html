Школа - здание предназначенное для прокачки навыков у спутников требующих Уважение с помощью бальзамов. Также обучать спутников можно в Казармах и Академии. Разница между этими постройками заключается в сложности строительства и базовом показателе преподавания:

  • Казармы - 1 Преп.
  • Школа - 10 Преп.
  • Академия - 30 Преп.

Показатель Преп. ограничивает максимальный навык спутника, который можно повышать в данной школе с помощью обучения. Например, если школа имеет показатель Преп. 42, то максимум в ней получится обучить спутника с навыка 42 до навыка 43, и для этого потребуется Бальзам с показателем 43 или выше. 

Эта постройка даёт большой  Престиж и, следовательно, подходит для того, чтобы оставлять своих спутников с требованием престижа работать на участке.

Формулы расчета показателей:

  • Уровень престижа равен (суме значений престижа от башен и укреплений) / 2
  • Уровень Преп. равен 10 + 0.9 *(среднее значение [среднее значение суммы местностей по каждой строительной части] + 0.6 * среднее значение престижа))

[среднее значение суммы местностей по каждой строительной части] вычисляется как ( + + + + + + + + ) / 9 с округлением в меньшую сторону. Например Школа из: Башня из известняка + Укрепление из известняка + Рама из ели + Матерчатая стена из льна + Матерчатая кровля из льна + Фундамент из ели будет давать 32 преподавания, чего достаточно для обучения Т3 спутников до максимума (33).
