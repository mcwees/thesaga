Как очистить кеш страницы

Как очистить кеш страницы:

в конец адреса страницы (URL) в строке браузера следует добавить

Шаблоны

Шаблоны позволяют задать некий шаблонный кусок текста, и далее подставлять его на любую страницу с помощью названия шаблона, заключенного в фигурные скобки. Это удобно тем, что информация начинает выглядеть единообразно, и чтобы отредактировать - достаточно исправить одно место, а не бегать по всей вики и править.

Примеры уже созданных шаблонов (используйте!):

Тип шаблона Текст шаблона для вставки Результат после вставки Комментарий
Для отраслей, профессий, предметов 
{{Травничество}}
 Травничество Добавляет ссылку на профессию с иконкой. Такие шаблоны заведены для каждой профессии и отрасли, см. примеры на главной странице. 
{{Травничество|24px|Травник}}
 Травник То же самое, что и предыдущий шаблон, но вы управляете:
  • Размером иконки (первый параметр после вертикальной черты, например - 24px)
  • Текстом ссылки (второй параметр после вертикальной черты, например - Травник) 
{{Травничество|24px||}}
 Добавляет только иконку профессии:
  • После первой вертикальной черты указываем размер иконки в px, например - 24px
  • После третьей вертикальной черты обязательно пишем "Иконка", как в примере 
{{Валун мрамора|24px}}
 Валун мрамора Добавляет только иконку предметы с ссылкой на общую группу:
  • После первой вертикальной черты указываем размер иконки в px, например - 24px 
{{Короткий плащ из хлопка|24px|Короткий плащ из хлопка отличного качества}}
 Короткий плащ из хлопка отличного качества Добавляет предметы с иконкой и названием качества, с ссылкой на общую группу:
  • После первой вертикальной черты указываем размер иконки в px, например - 24px
  • После второй вертикальной черты указываем альтернативное название, например - Короткий плащ из хлопка отличного качества
Для территорий 
{{Каменоломня у отвесных скал|0}}
 Каменоломня у отвесных скал, ( - 0) Вставляет название места, территорию, и уровень сложности места
  • Если после вертикальной черты ничего не указать - подставится уровень сложности заданный в шаблоне
  • Если указать - подставится указанное значение

Создание шаблонов

Коллекция шаблонов растет. Если вам какого-то не хватает, вы можете добавить его сами, для этого:

  • Пишем название шаблона в {{}}, на той странице где он нужен. Старайтесь придумать такое название, которое соответствует игровому названию
  • Нажмите "Предпросмотр страницы"
  • Снизу под окном редактирования вы увидите название своего шаблона написанное красным (так как его еще нет)
  • Откройте ссылку с названия шаблона в новом окне
  • Отредактируйте свой шаблон. Добавьте то, что должно подставляться вместо него:
Тип шаблона Текст шаблона Комментарий
Для отраслей, профессий, предметов 
[[Файл:Лидерство.png|{{{1|16px}}}]] {{#if:{{{3|}}}||
{{{3|[[Лидерство|{{{2|Лидерство}}}]]}}} }}<noinclude>[[Category:Шаблоны Навыки]]</noinclude>
(Всё в одну строчку) 		Все слова "Лидерство" в примере меняем на название вашего шаблона. Остальное не трогаем, и не думаем, что это значит ))
Для отраслей, профессий, предметов. дополнение от Димариуса и Кренделя 
[[Файл:Таинственный ящик.png|link=Предмет для задания|{{{1|24px}}}|Таинственный ящик]] {{#if:{{{3|}}}||
{{{3|[[Предмет для задания|{{{2|Таинственный ящик}}}]]}}} }}<noinclude>
[[Category:Шаблоны Предмет для задания]]</noinclude>
(Всё в одну строчку)
  • Шаблон:Таинственный ящик
В результате такой постановки получаем шаблон, с названием предмета и его картинкой, который ссылает на страницу типа, в данном случае - Предмет для задания. Текст и картинка работают как ссылки на страницу типа, всплывающие подсказки выдают текст с названием предмета.
Для территорий 
[[Каменоломня у отвесных скал]], ({{{{{1|Знание равнин}}}|24px||Иконка}} - {{{2|7}}})
 Подставляем свои значения:
  • Вместо "Каменоломня у отвесных скал" - название вашего шаблона
  • Вместо "Знание_равнин" - название необходимого знания по умолчанию для данной местности (задать две местности по умолчанию пока нельзя, думаем над этим)
  • Вместо "7" - уровень необходимого знания по умолчанию для данной местности (можно указать диапазон, например 10-20)
Для монстров 
{{Монстр|имя=Заяц|мини={{{мини|}}}|размер={{{1|}}}|подпись={{{2|не задана}}}}}
<noinclude>[[Category:Шаблоны Спутники животные]]</noinclude>
(Всё в одну строчку)
  • Вместо "Заяц" - имя монстра
  • Должны быть загружены картинки с именами вида "Заяц.png" и "Заяц мини.png"

Создание иконок для предметов

Наш многоуважаемый коллега NightCroucher написал программу для покраски ч/б изображений в необходимые цвета.
Вот сам ссылка на нее
Внутри архива вы обнаружите:

  • 2 папки - Pics и Result;
    • В папке Pics содержатся ч/б варианты иконок, на которые накладывается цвет.
      Если какой либо иконки нету, ее можно взять с русского офф сайта, с англ офф сайта или с паков, имеющихся у нас
    • В папку Result сохраняется окрашенная иконка с именем вида "Имя ч/б иконки_Имя схемы цвета"
  • файл colors.xml, в котором и прописаны все цвета
  • файл TherianSaga Colorer.exe, собственно сам исполняемый файл программы

Категории

Категорию (одну или несколько) можно присвоить к любой странице. И далее на странице категории будут выводиться все привязанные к ней статьи, автоматически. Это очень удобно, поэтому желательно всем своим статьям присваивать категории. Это делается тегом: 

[[Category:Название категории]]

Пока система категорий только формируется, но в любом случае, каждую статью можно соотнести с тем разделом меню слева, в котором она находится (Профессии, Квесты и так далее). Статьи, описывающие деятельность в определенной местности можно привязывать к этой местности. Например (Соколиния, Ланфар).
