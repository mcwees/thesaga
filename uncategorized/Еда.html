Еда используется для приручения животных, обмена с торговцами и изготовлении кормов.

Виды еды

Еда Где достать Для чего используется
Яйца
  • Продается в Журавлиной гавани у Джинили Гурэн за 1
  • Добывается навыком Животноводство при производстве кур
  • для найма Человек-троглодита
Морковь
  • Продается в Лесу Бериус у Лиоль дрессировщицы за 1
  • для приручения зайца
  • для приручения шумного ориктуса
  • для Черепахолев и ориктус (квест)
Яблоко
  • Продаётся в Саду тысячи плодов у Гертруды Орхидезии за 6
  • для приручения кабана
  • для приручения снежного лося
Овсянка
  • Продаётся на Ферме семьи Гурэн у Марлы Гурэн за 5
  • для приручения оленихи
  • для приручения оленя
  • для приручения дикого пони
  • для приручения зубра
  • для приручения снежного лося
Ромбулар
  • Продаётся в Журавлиной гавани у Миранды за 3
  • для Нехватка ромбулара (квест)
  • для фестиваля ромбулара
  • для найма Здоровяка
Еда для черепахольвов Производится в Доме на холме (A2)
  • для приручения молодого прирученного черепахолева
Зайчатина
  • Добывается навыком Охота при разделывании тушки зайца
  • Добывается навыком Животноводство при производстве кроликов
  • для приручения гульбора
  • для приручения рыси
  • для приручения филгара
  • для фестиваля ромбулара
Оленина
  • Добывается навыком Охота при разделывании туши оленихи, оленя лося
  • Продаёт Тамори Кантар из Таверны "Мокасин" в Громыхающем лесу в ограниченном количестве
  • для приручения медведя
  • для приручения волка
  • для приручения лесной виверны
  • для фестиваля ромбулара
Говядина
  • Добывается навыком Охота при разделывании туши зубра или туши белого зубра

  • для фестиваля ромбулара
Кабанятина
