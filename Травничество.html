Зелья

Рецепты зелий

Характеристики рецептов

Прежде чем изготовить зелье, травник должен выучить соответствующий рецепт.

Рецепт определяет:

  • тип зелья и, соответственно, набор навыков, на которые будет оказывать действие данное зелье;
  • время и затраты энергии (усилие), необходимые для приготовления зелья;
  • необходимые типы ингредиентов и их пропорции:
    • количество диких растений, метарастений и рун;
    • количество связывающих веществ;
    • тип ёмкости, в которую будет помещено зелье;
  • базовую сложность зелья (H0) — минимальный навык травничества, который необходим для создания зелья;
  • базовую длительность эффектов зелья (L0) — минимальную длительность эффектов зелья;
  • базовую токсичность зелья (Tx0) — минимальную токсичность зелья.

Свойства зелий

Общие сведения, формулы
Свойства зелья определяются ингредиентами, которые травник использовал при изготовлении данного зелья. При изготовлении зелья могут быть использованы любые ингредиенты и в любых сочетаниях. Единственное ограничение — количество и тип ингредиентов, которое может быть использовано.

От ингредиентов, применяемых при изготовлении зелья, зависят:

  • уровень травничества, который необходим для изготовления зелья;
  • сопротивление ингредиентов смешиванию, которое необходимо преодолеть для изготовления зелья;
  • длительность эффектов зелья;
  • токсичность зелья;
  • величины бонусов к навыкам персонажа.

Для лучшего понимания механики работы зелий см. общий обзор зелий.

Сложность зелья

Сложность зелья, то есть уровень травничества, который необходим для изготовления данного зелья, определяется по формуле:
H = H0 + Ii - 0,01×H0×Ii[1],
где:

  • H — результирующая сложность зелья;
  • H0 — базовая сложность зелья; данный параметр определяется рецептом;
  • Ii — самая высокая неизменность, которая встречается у смешиваемых ингредиентов.


Сопротивление смешиванию

Чтобы изготовить зелье, необходимо обеспечить в процессе его изготовления такую силу смешивания, которая могла бы преодолеть наивысшее сопротивление смешиванию, встречающееся среди используемых ингредиентов.

Сила смешивания, доступная при изготовлении зелья, рассчитывается по формуле:
F = Fh + 0,5×(Fs + Fv)
где:

  • F — результирующая сила смешивания, которая будет доступна при изготовлении зелья;
  • Fh — сила смешивания персонажа;
  • Fs — сила смешивания связывающего вещества, использованного при изготовлении зелья;
  • Fv — сила смешивания использованной ёмкости.

Сила смешивания персонажа (Fh) формируется из сил смешивания предметов экипировки, спутников и места, в котором проводится работа. Результирующая сила смешивания персонажа рассчитывается так же, как и результирующее значение его навыков. То есть компонент, имеющий наивысшую силу смешивания, даст вклад с коэффициентом 0,4 (40%), следующий по силе компонент даст вклад с коэффициентом 0,3 (30%) и т. д. Талисманы, специальный вид бижутерии, увеличивают силу смешивания персонажа без модификаторов.

Длительность эффектов зелья

Длительность эффектов зелья может быть рассчитана по формуле:
L = L0 + Li[1],
где:

  • L — результирующая длительность эффектов, которой будет обладать зелье;
  • L0 — базовая длительность эффектов; данный параметр определяется рецептом зелья;
  • Li — наибольшая длительность эффектов, которая встречается у смешиваемых ингредиентов.

Длительность эффектов зелья — условная величина. Для того чтобы вычислить время, в течение которого будет действовать зелье, необходимо воспользоваться следующей формулой:
T = 2 + 0.25×L,
где:

  • T — выраженное в часах время действия эффектов зелья;
  • L — длительность эффектов зелья.


Токсичность

Токсичность зелья может быть рассчитана по формуле:
Tx = Tx0 + Txi - 0,01×Tx0×Txi[1],
где:

  • Tx — результирующая токсичность, которой будет обладать зелье;
  • Tx0 — базовая токсичность; данный параметр определяется рецептом зелья;
  • Txi — наибольшая токсичность, которая встречается у смешиваемых ингредиентов.


Величины бонусов

Навыки персонажа, на которые будет воздействовать зелье, определяется его рецептом, а бонусы к этим навыкам зависят от ингредиентов, которые травник использовал при создании зелья. Чтобы определить величину бонуса к навыку, которым будет обладать зелье, необходимо воспользоваться формулой:
E = kt×(E0 + Ei - 0,01×E0×Ei)[1],
где:

  • E — результирующий бонус к навыку, которым будет обладать зелье;
  • kt — коэффициент (модификатор), зависящий от типа зелья: для лечебного зелья и зелья энергии равен 1,0, для прочих типов зелий равен 0,1;
  • E0 — базовый бонус к навыку;
  • Ei — бонус к навыку, зависящий от ингредиентов, использованных травником.


Чтобы определить бонус к навыку, который дадут смешиваемые ингредиенты (Ei), необходимо воспользоваться формулой:
Ei = kacc×Pacc + kact×Pact + kclm×Pclm + kenr×Penr + kfoc×Pfoc + kfrt×Pfrt + kinv×Pinv + kst×Pst,
где:

  • kacc, kact, kclm, kenr, kfoc, kfrt, kinv, kst — коэффициенты (модификаторы), применяемые соответственно к следующим свойствам ингредиентов: ускорение, возбуждение, успокоение, бодрость, концентрация, укрепляющее действие, воодушевление, стимулирующий эффект;
  • Pacc, Pact, Pclm, Penr, Pfoc, Pfrt, Pinv, Pst — наибольшие значения, встречающиеся среди всех смешиваемых ингредиентов, для ускорения, возбуждения, успокоения, бодрости, концентрации, укрепляющего действия, воодушевления, стимулирующего эффекта.
