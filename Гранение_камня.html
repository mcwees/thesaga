Гранение камня - один из навыков профессии Камень. Используется при гранении валунов и при резьбе по камню из каменных блоков.

Навык Гранение камня
Профессия Камень
Мастерские
  • Гранение камня
  • Резьба по камню
  • Гранение монолитов
  • Руническая резка
Максимально достижимый уровень навыка 87



Для изготовления нужного блока используется Валун соответствующего материла и один Резец, для резьбы из блоков помимо одного резца могут потребоваться дополнительные материалы, такие как Рукоятка. Чтобы изготовить любое изделие каменщику необходимо преодолеть Механическое сопротивление основного материала ( валуна или блока) с помощью Механической силы и иметь требуемый уровень навыка Гранение камня. Навык и сила рассчитываются по формуле, подробно описанной в разделе Как происходит расчет навыков?. Здесь отдельно отметим, что для повышения силы каменщику не обязательно в качестве инструмента использовать Каменотесное долото. Механической силой также обладает инструмент охотника - Охотничий нож. Таким образом, если у вас одеты Каменотесное долото и Охотничий нож, и у ножа механическая сила больше, чем у долото, то для расчёта навыка будет использоваться навык Гранение камня от долото, а для расчёта силы - Механическая сила от охотничего ножа

Мастерские навыка

Для каждого навыка существуют свои мастерские, в которых, используя данный навык, можно обрабатывать материалы или создавать уже готовую продукцию. Для гранения камня такими мастерскими являются Гранение камня и Резьба по камню.

Гранение камня

Мастерская, в которой вы можете обрабатывать валуны и получать из них блоки.
При огранке используются валун и резец, на выходе получаем блок и песок, того же типа, которого и валун.

  • Блоки используются в дальнейшей работе как каменщика (Резьба по камню, Каменная кладка), так и инженера (Машиенум).
  • Песок используется мастерами-литейщиками (Работа по стеклу) для выплавки стекла и алхимиками (Литрумы) для создания цементов.

Имение Мегалитическая стройка


В этой мастерской вам предоставляется возможность вырезать из камня всевозможные предметы, которые в дальнейшем используются мастерами иных профессий для создания более комплексных вещей, а также для обустройства домов.
Можно создавать следующие вещи:

  • Точильный камень - используется мастерами-ювелирами при огранки самоцветов, является расходным материалом. При огранке расходуется 1 точильный камень на 1 самоцвет.
  • Каменная палица - простенькая каменная булава.
  • Каменная оправа - заготовка для ювелиров.
  • Ступка и пестик - инструмент травников, используемый ими при варке зелий.
  • Каменная застежка - используется ювелирами при создании сложных украшений.
  • Каменная чашечка - раходный материал алхимиков для создания бальзамов.
  • Каменное ювелирное украшение - как и каменная застежка, нужна ювелирам при создании украшений.
  • Каменный боевой молот - серьёзное оружие с дробящим уроном.
  • Сланцевая дранка - используется плотниками для создания сланцевой кровли.
  • Каменная скульптура - элемент декора, используемый при строительстве зданий на вашем участке для поднятия уровня престижа.
  • Каменная отделка - используется в больших количествах при создании крутой мебели.
  • Каменная статуя - аналогична каменной скульптуре.
  • Каменный скипетр - лучшее оружие с дробящим уроном.

Прокачка навыка

В таблице собраны все изделия, создаваемые навыком Гранение камня, упорядоченные по возрастанию навыка. Для того, чтобы определить, как проще всего дальше прокачать навык, нужно найти в таблице одно из изделий с максимальной сложностью, которое вы можете изготовить. Если личный навык героя на 4 или больше превышает максимальную доступную сложность, следует обратиться к разделу Усиления навыка и каким-либо способом повысить свой навык. При выборе изделия из нескольких подходящих можно ориентировать на значение общего времени в таблице - это время, необходимое на добычу валунов, их огранку и изготовление самого изделия. То есть общее время характеризует сложность выполнения задачи.
