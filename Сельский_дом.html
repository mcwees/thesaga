Сельский дом - жилое здание. Тут вы можете отдохнуть с комфортом, и набраться больше сил, чем если бы вы спали в обычном лагере. 

Эта постройка хорошо подходит для того, чтобы оставлять своих спутников работать на участке.
Данное строение рекомендуется использовать, если вы собираетесь использовать Т3 спутников для добычи ресурсов.
Для достижения показателя комфорта  40 (требуемого для Т3) одним из элементов мебели рекомендуется использовать  Кровать из кедра сложного типа с постельными принадлежностями из хмеля или хлопка.

Формулы расчета показателей:

  • Уровень лидерства равен (самое высокое значение лидерства одного из компонентов)
  • Уровень престижа равен (самое высокое значение престижа одного из компонентов)
  • Уровень комфорта равен (самое высокое значений комфорта одного из компонентов мебели * 0,8) + 20
