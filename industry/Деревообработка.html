  • Штифты - деревянные скрепы - используемые для сборки предметов.
  • Дубина - толстая, тяжелая палка.
  • Посох - трость особого вида и назначения.
  • Рондаш - европейский щит конников раннего Средневековья.
  • Деревянная оправа - используется ювелирами при создании украшений.
  • Рукоятка - используется кузнецами при создании инструмента или оружия.
  • Палица - тяжелая дубинка с утолщенным концом.
  • Кол - палка, заточенная с одного конца.
  • Деревянная застежка - приспособление для соединения различных элементов одежды, обуви, предметов галантереи, используется ювелирами при создании украшений.
  • Деревянный круглый щит - вид военного защитного снаряжения, предназначенный для отражения атак холодного или стрелкового оружия.
  • Сабо - деревянные башмаки.
  • Киянка - столярный молоток из дерева твёрдых пород.
  • Деревянное ювелирное украшение - используется ювелирами при создании украшений.
  • Деревянная миска - предмет посуды глубокой формы, используется алхимиками при создании краски.
  • Деревянный каплевидный щит - тип щита, применявшегося в Европе в X—XIV веках, назван так за соответствующую форму.
  • Деревянный тарч - тип щита, применяемый в европейской армии с XIII по XVI век. Это выгнутые щиты различной формы, обычно имевшие "локтевое" крепление.
  • Деревянная павеза - тип щита, широко применявшийся пехотой в XIV—XVI веках. Щит был прямоугольной формы, нижняя часть могла иметь овальную форму.
  • Деревянная отделка - используется портными при создании одежды из разряда Высокой моды
