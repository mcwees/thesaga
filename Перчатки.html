Характеристики Клепаных кожаных перчаток напрямую зависят от использованной кожи и заклепок. Базовые значения защиты определяются кожей, а дополнительные (бонус) - характеристиками металла заклепок. Бонус линейно зависит от следующих характеристик:

  • Защита от тупых ударов - жесткость
  • Защита от проникающих ударов - твердость
  • Защита от рубящих ударов - плотность

Таким образом, наиболее выгодными металлами для изготовления Клепаных кожаных перчаток являются: Заклепки для доспеха из бронзы, Заклепки для доспеха из железа, Заклепки для доспеха из стали, Заклепки для доспеха из одиемеля, Заклепки для доспеха из крианрока, Заклепки для доспеха из крианделя. Таблицу с характеристиками приведём только для них и для базовых заклепок.
